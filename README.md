# Customer-churn-analysis-
## 📌 Project Title
End-to-End Customer Churn Analysis using Excel, SQL and Power BI

## 📝 Short Description / Purpose
This project focuses on analyzing customer churn to understand why customers leave and what factors influence their decision.
I worked on this project using an end-to-end approach — starting from data preparation, followed by SQL analysis, and finally building an interactive dashboard in Microsoft Power BI to visualize insights.

The main goal of this project is to identify high-risk customers and provide insights that can help improve customer retention.

## 🛠️ Tech Stack
- Microsoft Excel (Data Preparation)
- SQL (Data Analysis & Queries)
- Power BI (Data Visualization & Dashboard)

## 📂 Data Source
Telecom Customer Churn Dataset (Sample dataset used for learning and analysis)

## ⭐ Feature Highlights
- Created Age Groups for better demographic analysis
- Identified customers with low satisfaction score having highest churn rate
- Analyzed churn across contract types and tenure groups
- Built interactive dashboard with KPIs and filters
- Segmented high-risk customer groups

## 📊 Dashboard Features
- Total Customers KPI
- Churn Rate KPI
- Average Tenure
- Churn by Satisfaction Score
- Churn by Contract Type
- Churn by Age Group
- Churn by Payment Method

## 🚀 Key Insights
- Customers with low satisfaction score had highest churn rate
- Short tenure customers were more likely to churn
- Higher monthly charges increased churn risk
- Month-to-month contract customers were more likely to churn
